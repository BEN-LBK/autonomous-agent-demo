# 🎮 2048 网页游戏

经典的 2048 滑块益智游戏网页版实现。

## 🎯 游戏规则

1. 使用**方向键**（↑ ↓ ← →）移动所有方块
2. 相同数字的方块碰撞时会**合并**成更大的数字
3. 每次移动后会随机出现一个新的 2 或 4
4. 目标是合并出 **2048** 或更高的数字！
5. 当无法移动时，游戏结束

## 🚀 如何开始

### 方法一：直接打开（推荐）

双击 `index.html` 文件，在浏览器中打开即可开始游戏！

### 方法二：使用本地服务器

```bash
# 使用 Python 3
cd game
python3 -m http.server 8000

# 然后在浏览器中打开
# http://localhost:8000
```

## 🏆 高分技巧

1. 尽量把大数字固定在一个角落
2. 保持行或列的有序性
3. 不要让大数字分散在中间
4. 提前规划多步移动

## 📊 分数说明

- 合并两个 2 → 得 4 分
- 合并两个 4 → 得 8 分
- 合并两个 8 → 得 16 分
- 以此类推...
